# 🐾 Sanos y Salvos - Plataforma de Localización de Mascotas

La empresa **Sanos y Salvos** busca resolver el problema de la pérdida de mascotas mediante una solución tecnológica integral. Esta plataforma inteligente permite a los dueños y ciudadanos colaborar activamente en la localización, recuperación y actualización del estado de los animales perdidos.

## 🚀 Características Principales (Requisitos Funcionales)

* **Gestión de Mascotas:** Organización y registro detallado de las mascotas. Permite aplicar filtros específicos (por ejemplo: filtrar solo por "perros", raza, tamaño, etc.).
* **Sistema de Geolocalización:** Herramienta para ubicar mascotas en un mapa interactivo y registrar reportes del último lugar donde fue avistado el animal.
* **Motor de Coincidencias (Match):** Sistema inteligente que cruza los datos de las mascotas reportadas como perdidas con aquellas encontradas por ciudadanos, alertando si hay similitudes físicas.

## 📱 Accesibilidad (Requisitos No Funcionales)

* **Diseño Multiplataforma (Responsive):** El software es 100% accesible para todos los usuarios desde cualquier dispositivo. Ya sea a través de un teléfono celular o un PC de escritorio, cualquier persona puede acceder al servicio para encontrar o reportar mascotas de forma rápida y sencilla.
